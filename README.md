👑 Crowning Our Class (우리 반 시상식)

웹캠을 활용하여 실시간으로 학생들에게 재미있는 상을 수여하고 머리 위에 황금 왕관을 씌워주는 간단한 웹 기반 이벤트 도구입니다. 제미나이 AI와 대화하며 만든 교실 활동용 프로젝트입니다.

✨ 주요 기능

실시간 웹캠 연동: 브라우저의 카메라 권한만으로 별도의 설치 없이 즉시 실행 가능합니다.

랜덤 상 수여: 16가지 이상의 다양하고 재치 있는 상장 문구가 무작위로 나타납니다.

화려한 효과: 상 수여 시 '황금 왕관' 그래픽과 함께 축하용 꽃가루(Confetti) 애니메이션이 화면을 채웁니다.

깔끔한 UI: 시인성이 좋은 다크 모드 기반의 디자인과 직관적인 버튼 구성을 제공합니다.

🚀 사용 및 배포 방법

파일 업로드: 깃허브 저장소에 crowning_our_class.html 파일을 업로드합니다.

GitHub Pages 설정:

저장소의 Settings > Pages 메뉴로 이동합니다.

Branch를 main으로 설정하고 Save를 누릅니다.

접속: 생성된 공식 URL(https://www.google.com/search?q=https://%EC%95%84%EC%9D%B4%EB%94%94.github.io/%EC%A0%80%EC%9E%A5%EC%86%8C%EC%9D%B4%EB%A6%84)%EC%9D%84 통해 어디서든 접속할 수 있습니다.

🛠 기술 스택

언어: HTML5, CSS3, JavaScript (Pure JS)

그래픽: HTML5 Canvas API

브라우저 지원: Chrome, Edge, Safari 최신 버전 권장

📌 주의사항

카메라 하드웨어를 제어하므로 반드시 HTTPS 보안 연결이 필요합니다. (GitHub Pages 배포 시 자동 적용)

모바일 기기에서도 접속 가능하나, 가로 모드에서 더 쾌적하게 이용하실 수 있습니다.
